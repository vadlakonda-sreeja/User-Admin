# Admin Portal

The **Admin Portal** is a web-based interface that allows administrators to manage user data.  
It supports full **CRUD operations** (Create, Read, Update, Delete) and interacts with a REST API built using **JSON Server**.

---

## Features

- Add new users
- Update existing user details
- Delete users
- View all users
- Responsive user interface
- Simple and clean design

---

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (ES6)
- Backend: JSON Server (Mock REST API)
